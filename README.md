## Run Locally

Clone the project

```bash
  git clone https://github.com/bastilavarias/calculator-react.git
```

Go to the project directory

```bash
  cd calculator-react
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
